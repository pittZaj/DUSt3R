## 📖 使用步骤

### 1. 访问Web界面

在浏览器中打开: http://localhost:7868

### 2. 上传图像

- 点击"上传煤堆图像"区域
- 选择至少2张图像（推荐5-10张）
- 支持JPG、PNG等格式

### 3. 设置参数

**煤炭密度** (kg/m³):
- 原煤: 1200-1400
- 精煤: 1300-1500
- 默认: 1300

**图像尺寸** (px):
- 224: 快速但精度较低
- 512: 推荐（默认）
- 更大尺寸需要更多显存

### 4. 开始处理

- 点击"🚀 开始处理"按钮
- 等待处理完成（可能需要几分钟）

### 5. 查看结果

系统会显示：
- 📊 体积（立方米）
- ⚖️ 重量（吨）
- 🎯 置信度
- 📦 点云文件下载
- 📄 结果JSON下载

## 🔧 常见问题

### Q1: 服务无法启动？

**检查项**:
1. 确认conda环境已激活
2. 检查端口7868是否被占用
3. 查看日志: `tail -f /tmp/demo_output.log`

**解决方案**:
```bash
# 检查端口
netstat -tuln | grep 7868

# 如果被占用，停止旧进程
./stop_demo.sh

# 重新启动
./start_demo.sh
```

### Q2: 处理失败？

**可能原因**:
- 图像数量不足（<2张）
- 图像质量差
- GPU内存不足

**解决方案**:
- 增加图像数量
- 提高图像质量
- 降低图像处理尺寸

### Q3: 结果不准确？

**优化建议**:
- 增加图像数量（10-15张）
- 提高图像质量
- 使用已知体积的煤堆校准
- 调整煤炭密度参数

### Q4: 处理速度慢？

**优化方案**:
- 减少图像数量
- 降低图像尺寸（224px）
- 使用更强大的GPU

## 📊 结果说明

### 体积 (m³)
煤堆的估算体积，使用三种方法融合计算：
- Delaunay三角剖分
- 凸包法
- 体素化方法

### 重量 (吨)
根据体积和密度估算：
```
重量(吨) = 体积(m³) × 密度(kg/m³) / 1000
```

### 置信度 (%)
基于多种方法一致性的评分：
- **>80%**: 结果可靠
- **60-80%**: 结果较可靠
- **<60%**: 建议增加图像或检查质量
