## 📊 问题分析

### 用户反馈的问题

**现有方法的歧义**：
> "所有点云文件上的点垂直映射到地面这个平面后最外层点组成的图像存在多个被其包裹（紧密的贴合）的长方形，即长方形不唯一，所以其地面真实长度（米）也会响应造成误判。"

**核心问题**：
1. 当前使用**轴对齐边界框（AABB）**的长度作为标定基准
2. AABB的长度取决于坐标系方向
3. 如果煤堆是斜向的，AABB会比实际尺寸大
4. 用户输入的"地面真实长度"含义不明确

### 示意图

```
俯视图（投影到地面）：

方法1：轴对齐边界框（旧方法）
┌─────────────────┐
│    *   *   *    │  ← AABB长度取决于坐标轴方向
│   *  煤堆  *    │     如果煤堆斜向，会偏大
│  *    *    *    │
└─────────────────┘

方法2：最大跨度（新方法）
    *   *   *
   *  煤堆  *
  *────────*        ← 最远两点距离，唯一且准确
     *    *
```

## 🎯 优化方案

### 新方法：最大跨度法

**定义**：使用凸包顶点中距离最远的两个点之间的距离作为标定基准

**优势**：
1. **唯一性** ✅：最大跨度是唯一的，不依赖坐标系方向
2. **直观性** ✅：用户可以直接测量煤堆最远两点的实际距离
3. **准确性** ✅：避免了AABB的方向依赖问题
4. **旋转不变性** ✅：无论点云如何旋转，最大跨度保持不变

### 实现原理

```python
# 1. 将所有点投影到地面平面
projected_points = points - distances[:, np.newaxis] * normal

# 2. 计算2D凸包
hull_2d = ConvexHull(xy_points)
hull_points = xy_points[hull_2d.vertices]  # 凸包顶点

# 3. 计算凸包顶点间的所有距离
from scipy.spatial.distance import pdist, squareform
distances_matrix = squareform(pdist(hull_points))

# 4. 找到最大距离
max_span = distances_matrix.max()
max_dist_indices = np.unravel_index(distances_matrix.argmax(), distances_matrix.shape)
span_point1 = hull_points[max_dist_indices[0]]
span_point2 = hull_points[max_dist_indices[1]]
```

## 📝 使用指南

### 现场测量方法

1. **找到煤堆最远两点**：
   - 目视或使用激光测距仪
   - 找到煤堆边缘最远的两个点

2. **测量实际距离**：
   - 使用激光测距仪或卷尺
   - 测量这两点之间的直线距离（单位：米）

3. **输入系统**：
   - 在"边界与堆顶"界面
   - 输入框：**最大跨度的真实长度（米）**
   - 输入测量值

4. **系统自动计算**：
   - 缩放因子 = 真实长度 / 点云最大跨度
   - 所有尺寸（高度、面积、体积）自动缩放

### 注意事项

1. **测量精度**：
   - 测量误差会直接影响体积计算
   - 建议使用激光测距仪（精度±1cm）
   - 避免使用卷尺（误差较大）

2. **测量时机**：
   - 在拍摄点云照片的同时测量
   - 确保煤堆形状未发生变化

3. **参考点标记**：
   - 可在煤堆边缘放置标记物
   - 便于识别最远两点的位置

## 🔬 技术细节

### 算法复杂度

- **凸包计算**: O(n log n)
- **距离矩阵**: O(k²)，其中k为凸包顶点数（通常k << n）
- **总复杂度**: O(n log n + k²)

对于典型点云（1000-10000点），计算时间<0.1秒。

### 数值稳定性

使用`scipy.spatial.distance.pdist`计算距离：
- 内部使用优化的C代码
- 数值稳定性好
- 支持大规模点云

### 边界情况处理

1. **点云太少**（<3点）：
   - 无法计算凸包
   - 返回最大跨度=0
   - 提示用户检查点云

2. **共线点**：
   - 凸包退化为线段
   - 最大跨度=线段长度
   - 仍然有效

3. **单点**：
   - 最大跨度=0
   - 无法进行尺度标定
