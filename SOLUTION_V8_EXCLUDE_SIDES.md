# 物料堆地面识别 - 终极解决方案

## 问题根源发现

经过深入分析，我发现了问题的根本原因：

### 关键发现

使用RANSAC在原始点云上找到的**最大平面是侧面，不是地面**！

```
点云分析结果：
- 总点数: 1603
- 最大平面: 1023点 (63.8%)  ← 这是侧面！
- 平面倾角: 58.53°
- Z值跨度: 0.0809 m（几乎覆盖整个高度）

结论：最大的平面是倾斜的侧面，占据了63.8%的点
```

**这就是为什么之前所有方法都失败的原因**：
- 传统RANSAC会找到最大的平面（侧面）
- 即使加了覆盖率、中心性、法线验证，侧面仍然可能得分很高
- 因为侧面确实很大、很平坦、覆盖率高

## 解决方案：先排除侧面，再找地面

### 核心思路

**物料堆的几何特性**：
1. **侧面**：倾斜（30-60°）、面积最大（>60%）、Z值跨度大
2. **地面**：水平（<10°）、面积小（<10%）、Z值最低

**算法策略**：
1. 使用RANSAC找到最大的平面
2. 如果倾角>15°，认为是侧面，排除这些点
3. 在剩余点中重复步骤1-2，直到找到水平平面或点数过少
4. 如果剩余点数过少，直接在全局点云中取Z值最低的5%点
5. 在这些最低点上拟合平面，这就是地面

### 算法流程

```python
def _fit_ground_exclude_sides(cloud, distance_threshold):
    """先排除侧面，再找地面"""

    remaining_cloud = cloud
    excluded_planes = []

    # 最多迭代5次，排除倾斜平面
    for iteration in range(5):
        # 在剩余点中找最大的平面
        plane, inliers = remaining_cloud.segment_plane(...)
        angle = 计算倾角(plane)

        # 如果是水平平面（<15°），认为是地面
        if angle < 15.0:
            验证是否在底部
            return plane, inliers

        # 如果是倾斜平面（>15°），认为是侧面，排除
        if angle >= 15.0 and 占比 > 20%:
            excluded_planes.append(plane)
            remaining_cloud = 移除这些点
        else:
            break

    # 如果没有找到水平平面，在全局点云中取最低5%点
    bottom_points = 全局Z值最低的5%
    plane = 拟合平面(bottom_points)
    return plane
```

```

- 迭代2: 找到平面 325点 (76.8%), 倾角=63.7° → 排除（侧面2）
- 迭代3: 找到平面 93点 (94.9%), 倾角=55.8° → 排除（侧面3）

全局底部点策略：

最终结果：
✅ 地面倾角: 0.14°
✅ 地面点数: 59

```

## 为什么这次成功了？

### 1. 明确识别并排除侧面

**之前的方法**：
- 试图通过评分、约束来"避免"选择侧面
- 但侧面太大、太平坦，仍然可能得高分

**现在的方法**：
- **主动识别侧面**（倾角>15°，占比>20%）
- **明确排除侧面**（从点云中移除）
- 在剩余点中找地面，或直接用最低点

### 2. 符合物料堆的物理特性

**物料堆的真实情况**：
- 侧面是主导（占60-80%的点）
- 地面点很少（<10%）
- DUSt3R重建主要是表面点

**算法适配**：
- 不依赖"最大平面是地面"的假设
- 明确处理"侧面是最大平面"的现实
- 最后兜底策略：直接用最低点（物理上地面必然是最低的）

### 3. 多层防护

1. **第一层**：迭代排除倾斜平面
2. **第二层**：验证水平平面是否在底部
3. **第三层**：兜底策略 - 全局最低点

## 给用户的建议

### 如何判断算法是否正确？

1. **查看日志**：
   ```
   迭代1: 找到平面 XXX点, 倾角=XX° → 排除（侧面）
   迭代2: 找到平面 XXX点, 倾角=XX° → 排除（侧面）
   ...
   全局底部平面: XX点, 倾角=X.X°
   ```

2. **检查倾角**：地面倾角应该<5°

3. **检查地面位置**：地面应该在底部（Z值最低）

### 如果仍然不对怎么办？

1. **检查点云质量**：
   - 是否包含地面？
   - 地面点是否太少？
   - 点云是否严重倾斜？

2. **调整参数**：
   - 降低distance_threshold（更严格的平面拟合）
   - 调整预处理参数（保留更多点）

3. **提供反馈**：
   - 哪个文件测试失败？
   - 倾角是多少？
   - 日志显示了什么？
