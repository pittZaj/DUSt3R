# 煤堆体积测算系统 Demo - TRELLIS.2版本

## 项目概述

本项目使用微软开发的TRELLIS.2大型3D生成模型（4B参数）来实现从单张煤堆图片生成3D模型并计算体积的功能。

## 功能特点

### 1. TRELLIS.2模型集成

- **单张图片生成3D**: 从单张煤堆图片生成高质量3D模型
- **高分辨率重建**: 支持512³到1536³的分辨率
- **复杂拓扑支持**: 可以处理开放表面、非流形几何等复杂结构
- **PBR材质**: 完整的物理渲染材质支持

### 2. 体积计算

- 基于生成的3D网格自动计算体积
- 支持自定义尺度因子校准
- 根据煤炭密度计算重量

### 3. 可视化

- 生成可交互的3D模型（HTML格式）
- 支持旋转、缩放、平移
- 导出PLY格式的3D网格文件

### 4. 备用方案

- 如果TRELLIS.2模型加载失败，自动回退到简化的深度估计方法
- 确保系统在各种情况下都能运行

## 参数说明

### 输入参数

1. **煤堆图像**: 上传单张煤堆照片
2. **煤炭密度**: 1.0-2.0 吨/立方米（默认1.3）
3. **尺度因子**: 0.1-10.0（默认1.0）
   - 用于将模型单位转换为实际米数
   - 建议使用已知参照物进行校准
4. **使用TRELLIS.2**: 是否使用TRELLIS.2模型（默认是）

### 输出结果

1. **体积**: 立方米
2. **重量**: 吨
3. **3D网格文件**: PLY格式
4. **可交互3D模型**: HTML文件（可在浏览器中打开）

## 工作流程

1. **图像输入**: 用户上传煤堆图片
2. **3D生成**:
   - 如果启用TRELLIS.2: 使用模型生成高质量3D网格
   - 如果禁用或失败: 使用简化的深度估计方法
3. **体积计算**: 基于3D网格计算体积
4. **结果输出**:
   - 显示体积和重量
   - 生成可交互的3D可视化
   - 保存网格文件

## 技术架构

### TRELLIS.2模型

- **架构**: 4B参数的Diffusion Transformer
- **输入**: 单张RGB图片
- **输出**: O-Voxel表示的3D模型
- **特点**:
  - 使用Sparse 3D VAE进行16×空间下采样
  - 支持任意拓扑结构
  - 完整的PBR材质建模

### 体积计算方法

1. **网格提取**: 从TRELLIS.2生成的模型中提取顶点和面
2. **水密性检查**: 检查网格是否水密
3. **体积计算**:
   - 如果水密: 直接计算体积
   - 如果不水密: 填充孔洞或使用凸包
4. **尺度校准**: 应用尺度因子转换为实际单位

## 注意事项

### 1. 首次运行

- TRELLIS.2模型会自动从Hugging Face下载（约4GB）
- 首次运行需要较长时间
- 确保网络连接稳定

### 2. 精度说明

- 单张图片生成的3D模型精度有限
- 体积计算结果为预估值
- 建议使用已知参照物进行校准
- 生产环境建议使用多视角图像或激光雷达

### 3. GPU使用

- 确保使用正确的GPU（CUDA_VISIBLE_DEVICES=1）
- 监控显存使用情况
- 如果显存不足，可以降低分辨率

## 故障排除

### 问题1: TRELLIS.2模型加载失败

**解决方案**:
- 检查网络连接
- 手动下载模型权重
- 使用备用方法（自动回退）

### 问题2: CUDA内存不足

**解决方案**:
- 降低输入图片分辨率
- 使用较低的生成分辨率
- 关闭其他GPU程序

### 问题3: 依赖缺失

**解决方案**:
```bash
conda activate TRELLIS
pip install <缺失的包>
```
