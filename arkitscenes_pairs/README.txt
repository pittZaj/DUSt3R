This archive contains the list of image pairs necessary to prepare the ARKitScenes dataset for training DUSt3R.

You can find more information at https://github.com/naver/dust3r

ARKitScenes is a large RGB-D dataset of indoor scenes. It is licensed under the Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International Public License. See https://github.com/apple/ARKitScenes/ for more details.
