## 二、开源技术调研与分析

### 2.1 3D重建技术方案对比

#### 方案一：传统SfM + MVS管线（COLMAP + OpenMVS）

**技术栈**：
- [COLMAP](https://colmap.github.io/)：Structure-from-Motion
- [OpenMVG](https://github.com/openMVG/openMVG)：Multiple View Geometry
- [OpenMVS](https://github.com/cdcseacave/openMVS)：Multi-View Stereo

**核心思想**：
1. **特征提取与匹配**：从多张图像中提取SIFT/ORB等特征点，进行特征匹配
2. **相机位姿估计**：通过Bundle Adjustment优化相机内外参数
3. **稀疏重建**：生成稀疏点云
4. **稠密重建**：通过立体匹配生成稠密点云
5. **网格重建**：从点云生成三角网格
6. **体积计算**：基于网格模型计算体积

**优势**：
- ✅ 技术成熟，经过大量实际应用验证
- ✅ 开源免费，社区活跃，文档完善
- ✅ 精度高，适合工业级应用
- ✅ 支持大规模场景重建

**劣势**：
- ❌ 需要相机标定和位姿估计，流程复杂
- ❌ 对图像质量要求高，纹理不足时效果差
- ❌ 计算耗时较长，不适合实时处理
- ❌ 需要大量重叠图像

**适用性分析**：
- **适用场景**：离线处理、高精度要求、有充足时间进行数据采集
- **本项目适配度**：⭐⭐⭐⭐ (4/5)
- **推荐用途**：作为基准方案，用于验证和对比

**任务3.3：煤堆分割**

```python
def segment_coal_pile(pcd, ground_height=0.0, margin=0.05):
    """
    从点云中分割出煤堆部分
    """
    points = np.asarray(pcd.points)
    
    # 1. 高度过滤：去除地面点
    mask = points[:, 2] > (ground_height + margin)
    
    # 2. 聚类分析：分离不同煤堆
    pcd_filtered = pcd.select_by_index(np.where(mask)[0])
    labels = np.array(pcd_filtered.cluster_dbscan(
        eps=0.1, min_points=10
    ))
    
    # 3. 选择最大聚类（主煤堆）
    max_label = labels.max()
    largest_cluster = labels == np.argmax(np.bincount(labels[labels>=0]))
    
    coal_pile_pcd = pcd_filtered.select_by_index(
        np.where(largest_cluster)[0]
    )
    
    return coal_pile_pcd

# 分割煤堆
```

#### 阶段四：体积计算实现（3-4天）

**任务4.1：尺度恢复**

由于MUSt3R输出的点云是相对尺度，需要恢复真实尺度：

```python
def recover_scale(pcd, reference_object_points, real_distance):
    """
    通过参考物体恢复真实尺度
    
    参数:
        pcd: 点云
        reference_object_points: 参考物体在点云中的点索引
        real_distance: 参考物体的真实尺寸（米）
    """
    ref_points = np.asarray(pcd.points)[reference_object_points]
    
    # 计算参考物体在点云中的尺寸
    measured_distance = np.linalg.norm(
        ref_points[0] - ref_points[1]
    )
    
    # 计算缩放因子
    scale_factor = real_distance / measured_distance
    
    # 缩放点云
    pcd.scale(scale_factor, center=pcd.get_center())
    
    return pcd, scale_factor
```

```python
import numpy as np

def calculate_volume_delaunay(pcd, ground_height=0.0):
    """
    使用Delaunay三角剖分计算体积
    """
    points = np.asarray(pcd.points)
    
    # 添加地面点（形成封闭体）
    # 获取煤堆底部边界
    bottom_points = points[points[:, 2] < (ground_height + 0.1)]
    
    # 投影到地面
    ground_points = bottom_points.copy()
    ground_points[:, 2] = ground_height
    
    # 合并点云
    all_points = np.vstack([points, ground_points])
    
    # Delaunay三角剖分
    tri = Delaunay(all_points)
    
    # 计算每个四面体的体积
    def tetrahedron_volume(a, b, c, d):
        return abs(np.dot(a - d, np.cross(b - d, c - d))) / 6.0
    
    total_volume = 0
    for simplex in tri.simplices:
        pts = all_points[simplex]
        # 只计算包含地面以上点的四面体
        if np.any(pts[:, 2] > ground_height):
            total_volume += tetrahedron_volume(*pts)
    
    return total_volume

# 计算体积
volume_m3 = calculate_volume_delaunay(coal_pile)
print(f"煤堆体积: {volume_m3:.2f} 立方米")
```

```python
from scipy.spatial import ConvexHull

def calculate_volume_convex_hull(pcd, ground_height=0.0):
    """
    使用凸包计算体积（适用于形状较规则的煤堆）
    """
    points = np.asarray(pcd.points)
    
    # 添加地面点
    x_min, x_max = points[:, 0].min(), points[:, 0].max()
    y_min, y_max = points[:, 1].min(), points[:, 1].max()
    
    # 在底部创建网格点
    ground_grid = np.array([
        [x, y, ground_height]
        for x in np.linspace(x_min, x_max, 20)
        for y in np.linspace(y_min, y_max, 20)
    ])
    
    all_points = np.vstack([points, ground_grid])
    
    # 计算凸包
    hull = ConvexHull(all_points)
    
    return hull.volume
```

```python
def calculate_volume_voxel(pcd, voxel_size=0.05, ground_height=0.0):
    """
    使用体素化方法计算体积
    """
    # 体素化
    voxel_grid = o3d.geometry.VoxelGrid.create_from_point_cloud(
        pcd, voxel_size=voxel_size
    )
    
    # 统计地面以上的体素数量
    voxels = voxel_grid.get_voxels()
    count = 0
    for voxel in voxels:
        center = voxel_grid.get_voxel_center_coordinate(voxel.grid_index)
        if center[2] > ground_height:
            count += 1
    
    # 计算体积
    volume = count * (voxel_size ** 3)
    
    return volume
```

```python
def calculate_volume_ensemble(pcd, ground_height=0.0):
    """
    使用多种方法计算体积并融合
    """
    # 方法1：Delaunay
    v1 = calculate_volume_delaunay(pcd, ground_height)
    
    # 方法2：凸包
    v2 = calculate_volume_convex_hull(pcd, ground_height)
    
    # 方法3：体素化
    v3 = calculate_volume_voxel(pcd, voxel_size=0.05, ground_height)
    
    # 加权平均（可根据实际情况调整权重）
    volume = 0.4 * v1 + 0.3 * v2 + 0.3 * v3
    
    # 计算置信度（基于方法间的一致性）
    std = np.std([v1, v2, v3])
    confidence = 1.0 / (1.0 + std / volume)
    
    return {
        'volume': volume,
        'confidence': confidence,
        'methods': {
            'delaunay': v1,
            'convex_hull': v2,
            'voxel': v3
        }
    }
```

## 四、关键技术难点与解决方案

### 4.1 难点一：无相机标定的3D重建

**问题描述**：
- 现场摄像头安装后难以精确标定
- 传统SfM方法需要相机内外参数

**解决方案**：
- ✅ **采用MUSt3R**：无需标定，直接从图像重建
- ✅ **参考物体法**：使用已知尺寸的参考物体恢复尺度
- ✅ **自标定方法**：如果需要，可以使用棋盘格等方法进行简易标定

### 4.2 难点二：纹理不足的煤堆表面

**问题描述**：
- 煤堆表面纹理单一，特征点少
- 传统特征匹配方法效果差

**解决方案**：
- ✅ **MUSt3R的优势**：基于深度学习，对纹理不足场景有较强鲁棒性
- ✅ **图像增强**：对比度增强、边缘增强
- ✅ **结构光辅助**：可选方案，投射结构光增加纹理
- ✅ **多模态融合**：结合RGB和深度相机

### 4.3 难点三：现场环境干扰

**问题描述**：
- 灰尘、雾气影响图像质量
- 光照变化（白天/夜晚、阴天/晴天）
- 运动模糊（传送带、车辆）

**解决方案**：
- ✅ **图像预处理**：去雾算法、降噪、对比度增强
- ✅ **工业级摄像头**：防尘防水、宽动态范围
- ✅ **补光设备**：夜间或光线不足时使用
- ✅ **关键帧选择**：避免运动模糊帧

### 4.4 难点四：实时性要求

**问题描述**：
- 需要实时或准实时处理
- MUSt3R推理耗时较长

**解决方案**：
- ✅ **模型优化**：ONNX转换、TensorRT加速、量化
- ✅ **增量更新**：仅处理变化区域
- ✅ **定时处理**：不需要每帧处理，可以每5-10分钟处理一次
- ✅ **边缘-云协同**：简单场景边缘处理，复杂场景云端处理

### 4.5 难点五：体积计算精度

**问题描述**：
- 点云质量影响体积计算精度
- 不同算法结果可能差异较大

**解决方案**：
- ✅ **多方法融合**：Delaunay、凸包、体素化多种方法结合
- ✅ **置信度评估**：基于方法一致性评估结果可信度
- ✅ **现场校准**：使用已知体积的煤堆进行校准
- ✅ **持续优化**：收集实际数据不断优化算法
